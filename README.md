# Simple Snake Game 🐍

A classic Snake game built using Python's built-in `turtle` graphics library. This project demonstrates basic game loops, coordinate systems, and event handling in Python.

## Features
* **Smooth Movement:** Controlled via keyboard arrow keys.
* **Randomized Food:** Food spawns at random locations upon being eaten.
* **Boundary Detection:** The game resets if the snake hits the wall.
* **Lightweight:** No external libraries required (uses standard Python libraries).
